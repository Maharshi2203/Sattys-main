## Project Summary
A premium food ingredients delivery application for Satty's, based in Gujarat. The project features a high-end landing page with video backgrounds, a product catalog with categorization, and an administrative dashboard for managing products, categories, and customer inquiries.

## Tech Stack
- **Framework**: Next.js 15 (App Router)
- **Language**: TypeScript
- **Styling**: Tailwind CSS, Framer Motion
- **Database**: Supabase (PostgreSQL)
- **Authentication**: Custom JWT-based admin authentication
- **Icons**: Lucide React, Tabler Icons

## Architecture
- `src/app`: Application routes and API endpoints
- `src/components`: Reusable UI components
- `src/lib`: Shared utilities and type definitions
- `src/hooks`: Custom React hooks

## User Preferences
- High-quality video backgrounds for the home page.
- Elegant, premium aesthetic (black/gold/white).

## Project Guidelines
- No comments in code unless requested.
- Use Supabase for database operations.
- Admin routes are protected and located under `/admin`.

## Common Patterns
- API routes use standard Next.js route handlers.
- UI components leverage Radix UI primitives and Tailwind CSS.
